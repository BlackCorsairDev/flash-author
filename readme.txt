== Description ==
This is a plugin to add a nice .gif background image to the author.

== Installation ==
1. Upload `flash-author` folder to your WordPress plugins directory (typically 'wp-content/plugins')
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Users > Your Profile and edit the author flash name.
4. Click "Update Profile" or "Update User"
